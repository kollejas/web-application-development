Web Basics

Web Application Development is a course at the study degree program Mobile Software Development and we will cover different topics in Web Engineering e.g. HTML, CSS, JS, ...

HTML

We are using HTML to create our document structure, where we will include content.

HTML is only to represent content and not style. We are able to add e.g. strong-Tags which will be displayed as bold elements, but strong is be used to focus / point important details. bold-tag specifies text to display bold font without any special importance.
Those two tags have different meanings for Search Engine Optimization, so take care how and when to use which Tag.

CSS

CSS is responsible to beautify and style our structured content.

It's easily possible to change style for different use cases, e.g. to display / style the HTML content for desktop or mobile usage.

JS

We will add special behaviour to our website with JS.

Interactive websites, Animations, Data Manipulation, ... will be possible without complete reloading of the previously loaded webpage.
